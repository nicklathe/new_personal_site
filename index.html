Cnvrge

I wanted to simplify the process of trying to find a place for two friends to meet for lunch. This app uses the geolocation of each user, and returns a list of up to 20 restaurant suggestions, which they can vote yes or no on, one at a time. It's built in Node.js with a MongoDB database, and features AngularJS and Socket.io for a dynamic real-time user experience. Github repo.

Seattle Brewery Finder

This is an app to help you find breweries in Seattle. You can create an account, and save your favorite ones for remembering later. It is written in Node.js, with Express.js. as the framework. I used two API's: Mapbox for the map, and BreweryDB for brewery information. The app works with a Postgres database, and features user authentication using bcrypt. Github repo.

Seattle Trading Post

I got to build this app with 3 other people in GA's WDI. We had a week to dream up, wireframe, and build the entire app. We all worked on both front end and back end aspects of the site. It currently uses 6 models and multiple API's, including Venmo and Cloudinary. Github repo.
